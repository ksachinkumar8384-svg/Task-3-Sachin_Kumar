🏧 ATM Interface (Java)

📌 Project Overview

The ATM Interface is a Java-based console application that simulates the basic functionalities of an Automated Teller Machine (ATM). The project allows users to perform essential banking operations such as checking account balance, depositing money, and withdrawing funds through a simple menu-driven interface.

This project is built using Object-Oriented Programming (OOP) principles and helps developers understand classes, objects, encapsulation, methods, and input validation in Java.


---

🚀 Features

Check account balance

Deposit money into the account

Withdraw money from the account

Validate transaction amounts

Prevent withdrawals when funds are insufficient

Interactive menu-driven interface

Display transaction status messages
